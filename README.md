# 📊 Bank Loan EDA & Regression Modeling

## 🔍 Project Overview
This project performs **Exploratory Data Analysis (EDA)** and **Regression Modeling** on a bank loan dataset to support data-driven credit risk assessment.

The goal is to analyze customer attributes, identify key risk indicators, and build a regression model to predict loan-related outcomes.

---

## 📁 Dataset
- Records: 2,500+ customers
- Data includes demographic, financial, and loan-related features

---

## ⚙️ Key Tasks Performed
- Data cleaning and preprocessing  
- Handling missing values and outliers  
- Feature correlation analysis  
- Exploratory Data Analysis (EDA)  
- Ordinary Least Squares (OLS) Regression modeling  

---

## 📈 Model Performance
- **Model:** OLS Regression  
- **R² Score:** 0.77  
- **MAPE:** 12.29%  

---

## 🔑 Key Insights
- Strong correlation observed between income, credit history, and loan approval
- Outlier treatment improved model stability
- Regression model supports effective credit risk assessment
